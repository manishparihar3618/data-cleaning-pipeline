# Data Cleaning Pipeline

This project demonstrates a simple data cleaning workflow using Python and Pandas.


## What was done
- Loaded raw student performance data
- Removed duplicate records
- Handled missing values using column-wise mean
- Saved the cleaned data to a new CSV file


## Files
- raw_data.csv – original dataset
- cleaned_data.csv – cleaned dataset
- clean_data.ipynb – data cleaning logic

##Tools Used
- Python
- Pandas
